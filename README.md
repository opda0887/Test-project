✍️**功能說明(Features)**:

1. 輸入「時事」後，會告訴您最近的相關新聞或頭條。

2. 輸入「財經」後，會告訴您最近的相關財經資訊。

3. 輸入「音樂」後，會告訴您最近的華語熱門排行。

4. 還是一隻稱職的聊天機器人喔 😆

5. 1~3 功能可以點及下方對應按鈕來快速查詢
